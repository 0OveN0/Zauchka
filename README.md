Для локального запуска сайта необходимо скачать файл FileLocalStart.zip, распаковать его на вашем ПК и запустить файл Start.bat
После успешного запуска, окно консоли выдаст ссылку на локальный сервер сайта
Перейдите по ней в браузере
Готово!
База данных также присутствует и изменяется локально на вашем ПК
Для доступа к админ-панели логин и пароль: admin
